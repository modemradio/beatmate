Fonctionnalites detaillees

Bibliotheque et Import

📚

Bibliotheque professionnelle

14 colonnes : Titre, Artiste, Album, BPM, Key, Camelot, Energy, Duree, Genre, Note, Cues, Stems, Annee, Plays. Tri sur toutes les colonnes. Recherche full-text instantanee.

📥

Import multi-source

Drag & drop, parcourir fichiers/dossiers. Formats : MP3, WAV, FLAC, AIFF, AAC, OGG, M4A, WMA. Import depuis Rekordbox (XML), Serato, Traktor (NML), VirtualDJ, Engine DJ, djay Pro.

🎧

Lecture des tags ID3

BPM, Key, Artist, Album, Genre, Year, Comment lus automatiquement via TagLib. Les donnees analysees par d'autres logiciels (Rekordbox, VirtualDJ) sont recuperees.

🔍

Filtres avances

Filtrez par BPM range, Key (Camelot), Genre, Energy (1-10), Rating (etoiles). Smart Playlists avec regles automatiques (AND/OR).

Analyse Audio

🎶

Detection BPM

Algorithme hybride (autocorrelation + FFT). Precision haute. Mode Normal (tempo fixe) ou Dynamique (tempo variable). Plage configurable (60-200 BPM).

🎹

Detection Key

Analyse chromatique + classification ML. Notation Camelot (1A-12B), Open Key, ou classique (Am, C, etc.). Compatible Mixed In Key.

⚡

Energy (1-10)

Score d'energie comme Mixed In Key. Prend en compte le loudness, le rythme et le contenu spectral. Ideal pour planifier la progression d'un set.

🌐

Waveform RGB

Waveform multi-bande coloree (bass=rouge, mid=vert, treble=bleu) comme Rekordbox. Cache pour chargement instantane.

📈

Structure musicale

Detection automatique : Intro, Verse, Chorus, Build, Drop, Bridge, Outro. Utilisee pour le placement intelligent des Hot Cues.

🤖

Stems IA (Demucs)

Personal+

Separation en 4 stems : Vocals, Drums, Bass, Other. Reseau neuronal Demucs. Premiere separation ~30s, ensuite cache.

Hot Cues

🎯

8 Hot Cues par piste

Placement manuel ou automatique IA. 16 couleurs DJ (palette Rekordbox/Serato). Snap-to-beat grid. Renommage et reorganisation.

🤖

Auto-generation IA

L'IA place automatiquement 8 cues : Mix-in, Verse, Buildup, Drop 1, Breakdown, Drop 2, Pre-outro, Mix-out. Basee sur l'analyse spectrale et l'energie.

📤

Export DJ Software

Export vers Rekordbox (XML), Serato (Markers2), Traktor (NML CUE_V2), VirtualDJ (POI), Engine DJ. Couleurs et positions preservees.

Preparation de Set

🎵

Score de compatibilite

Score 0-100 entre chaque paire de morceaux : Key Camelot (30pts), BPM (25pts), Energy flow (20pts), Genre (15pts), Variete (10pts).

📈

Courbe d'energie

Visualisation Studio-style de la progression d'energie du set. Gradient vert-jaune-rouge. Score global du set.

🤖

IA Auto-Arrange

Tri automatique par BPM, Key harmonique, ou Energy. Fill gaps : l'IA insere des morceaux pour combler les transitions faibles.

💾

Persistance

Le set est sauvegarde automatiquement en JSON. Drag & drop pour reordonner. Export M3U, JSON, HTML.

Preparation Soiree

🎆

Planification par phases

5 phases par defaut : Accueil, Cocktail, Diner, Danse, Fin. Chaque phase a genre, BPM cible, energy, duree et couleur.

📅

Timeline interactive

Timeline visuelle avec drag-resize des phases. Presets d'energie : Classic DJ Set, Double Drop, Marathon, Progressive, Wedding.

🤖

IA Auto-Fill

Remplissage automatique des phases avec les morceaux les plus adaptes (genre, BPM, energy). Compatibilite harmonique Camelot.

☑

Checklist DJ

5 items : Sono, Lumieres, Rider, Backup USB, Setlist imprimee. Sauvegarde persistante.

Technologies

JUCE 8.0.4

Framework audio/UI multiplateforme. Lecture audio temps reel. Interface graphique GPU-acceleree.

TagLib 2.0

Lecture/ecriture tags ID3v2 (MP3, FLAC, WAV, AIFF, OGG, M4A). BPM, Key, Artist, Album, Genre, Year.

Demucs (Meta AI)

Reseau neuronal pour la separation de stems. 4 bandes : vocals, drums, bass, other.

ONNX Runtime

Inference ML locale (pas de cloud). Classification genre, mood, energie.

RubberBand

Time-stretching et pitch-shifting qualite studio. Preserve la qualite audio.

SQLite3

Base de donnees locale. Recherche full-text (FTS5). Migration automatique des schemas.

Systeme de licence

1 licence = 1 machine. Chaque PC necessite sa propre cle de licence, liee au hardware ID unique de la machine.

La licence est verifiee localement (pas de connexion internet requise apres activation). Le fichier de licence est chiffre et signe avec le hardware ID.

Format des cles : XXXXX-XXXXX-XXXXX-XXXXX (prefixe P=Personal, R=Professional, E=Premium; 2e caractere M=Mensuel, A=Annuel)

Anti-piratage : Trial chiffre avec hardware ID, detection de manipulation d'horloge, fichier de licence signe SHA256.

Questions frequentes

Puis-je utiliser la meme licence sur plusieurs PC ?

Non. Chaque licence est liee a une seule machine via son hardware ID (CPU + carte mere + disque). Si vous avez 2 PC, vous avez besoin de 2 licences.

Que se passe-t-il apres les 7 jours d'essai ?

La lecture audio est limitee a 5 minutes par morceau. Un message s'affiche quand la limite est atteinte. Les modules Live et Studio sont bloques. Toutes les autres fonctionnalites restent accessibles.

Puis-je passer d'un forfait a l'autre ?

Oui. Entrez simplement la nouvelle cle dans Parametres > Licence. L'ancienne cle est desactivee automatiquement.

Comment activer ma licence ?

Allez dans Parametres > Licence > entrez votre cle > cliquez "Activer". La licence est verifiee et stockee localement sur votre machine.

Les mises a jour sont-elles incluses ?

Oui, toutes les mises a jour sont incluses pendant la duree de votre abonnement. Les forfaits Professional et Premium recoivent les mises a jour en avance (acces beta).

Mes fichiers audio sont-ils modifies ?

Non. BeatMate ne modifie jamais vos fichiers audio. Toutes les donnees (BPM, Key, cues, playlists) sont stockees dans la base de donnees locale.

Quels formats audio sont supportes ?

MP3, WAV, FLAC, AIFF, AAC, OGG, M4A, WMA, ALAC. Tous les formats standards sont supportes.

Comment importer depuis Rekordbox ?

BeatMate detecte automatiquement Rekordbox. Si la base est chiffree (Rekordbox 6+), exportez en XML depuis Rekordbox (File > Export collection as XML). BeatMate lit l'XML avec tous les BPM, Key, cues et playlists.

La separation de stems fonctionne-t-elle hors ligne ?

Oui. Le modele Demucs tourne localement sur votre CPU. Aucune connexion internet n'est necessaire. Premiere separation ~30 secondes, ensuite les resultats sont caches.

Que contient le module DJ et Studio ?

Mix DJ (2/4 decks, beat sync, crossfader, EQ), montage audio (timeline, clips, automation), remix/mashup/medley/megamix, effets DSP (reverb, delay, filter, etc.), automix IA, export audio et video.
