Bibliothèque intelligente

Gestion complète de votre collection musicale

Gratuit PRO

La bibliothèque BeatMate V11 gère plus de 13 000 tracks avec une fluidité remarquable. Basée sur TagLib pour la lecture native des métadonnées, elle offre 33 colonnes personnalisables incluant BPM, Key, Genre, Artist, Energy, Rating, Comment, Label, Year et bien plus. L'arborescence PC intégrée permet de parcourir vos dossiers directement sans import préalable.

✓ 33 colonnes de métadonnées personnalisables (tri, masquage, réorganisation)
✓ Filtrage avancé par BPM, Key, Genre, Artist, Energy, Rating, Year, Label
✓ Arborescence PC intégrée pour parcourir les dossiers sans import
✓ Système de rating par étoiles (0-5) avec raccourcis clavier
✓ Lecture native TagLib : MP3, FLAC, WAV, AAC, AIFF, OGG, WMA
✓ Recherche instantanée multi-champs avec auto-complétion
✓ Gestion de base de données SQLite optimisée pour 13 000+ tracks
✓ Album art intégré et cover art en haute résolution

Analyse audio avancée

Détection précise et automatisée

PRO PREMIUM

Le moteur d'analyse de BeatMate V11 combine plusieurs algorithmes pour une précision maximale. La détection BPM hybride croise les résultats de multiples méthodes. L'analyse Key supporte les notations Camelot et Open Key. Le calcul d'Energy sur une échelle 1-10 permet de construire des courbes de set cohérentes. Les Waveforms RGB haute résolution offrent un aperçu visuel détaillé.

✓ Détection BPM hybride multi-algorithmes (précision ±0.1 BPM)
✓ Analyse Key : notation Camelot Wheel et Open Key simultanées
✓ Calcul Energy (1-10) basé sur l'analyse spectrale et la dynamique
✓ Waveform RGB haute résolution avec découpage fréquentiel (low/mid/high)
✓ Beatgrid IA : détection automatique des downbeats et mesures
✓ Analyse batch : traitement de bibliothèques entières en arrière-plan
✓ Correction manuelle BPM/Key avec lock pour éviter l'écrasement

✓ Courbe d'énergie interactive : visualisez et ajustez le flow du set

✓ Export M3U / JSON / HTML avec métadonnées complètes du set
✓ Durée cible configurable : l'IA ajuste le nombre de tracks automatiquement

Normalisation LUFS

Volume homogène pour un mix sans surprise

PRO

La normalisation LUFS (Loudness Units relative to Full Scale) garantit un volume perçu homogène entre toutes vos tracks. Fini les sauts de volume en plein mix. BeatMate analyse le loudness intégré de chaque fichier et calcule le gain à appliquer pour atteindre la cible LUFS configurée.

✓ Analyse Loudness intégrée (Integrated LUFS) conforme EBU R128
✓ Cible LUFS configurable (défaut : -14 LUFS pour le DJ)
✓ Calcul de gain automatique par track avec true peak limiter
✓ Application non-destructive : le fichier original n'est jamais modifié
✓ Traitement batch pour normaliser toute la bibliothèque en un clic

DJ et Studio

Module externe de production et mix

PREMIUM NEW

Le module DJ et Studio est un outil externe qui transforme BeatMate en véritable station de production. Créez des remix, mashups et medleys directement depuis votre bibliothèque. Interface configurable de 2 à 4 decks avec mixage, EQ 3 bandes, effets et enregistrement intégré.

✓ Interface 2 ou 4 decks avec crossfader et EQ 3 bandes
✓ Remix en temps réel : découpe, boucle, réarrangement au beat
✓ Mashup intelligent : combinaison automatique des stems
✓ Medley builder : enchaînements automatiques avec transitions
✓ Effets intégrés : reverb, delay, filter, flanger, phaser
✓ Enregistrement et export audio haute qualité (WAV, FLAC, MP3)
